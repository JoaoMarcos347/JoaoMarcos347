## Projetos em Destaque

### [ImportaREST GO](https://github.com/JoaoMarcos347)

Plataforma de automacao fiscal que converte NFS-e (XMLs) em arquivos TXT para importacao no ISS.NET.

- Parsing de **50+ campos** extraidos de XMLs fiscais
- Classificacao de itens LC 116 via **GPT-4o-mini + N8N** com Review Gate de confianca adaptativa
- Suporte **multi-municipio** e **multi-tenant**
- Dual deployment **desktop/web** com fidelidade byte-a-byte validada por regressao

**Stack:** FastAPI | Python | Supabase | pgvector | JWT

> Repositorio privado — disponivel para avaliacao mediante solicitacao.

---

### [CineVault](https://github.com/JoaoMarcos347/CineVault)

App de colecao pessoal de filmes com foco em experiencia do usuario e acessibilidade.

- Integracao com **OMDb API** para busca de filmes
- **localStorage** para persistencia local
- Skeleton loading, undo com timeout
- Acessibilidade: `aria-labels`, `prefers-reduced-motion`

**Stack:** HTML | CSS | JavaScript

---

### [NexoPay](https://github.com/JoaoMarcos347/NexoPay)

Checkout page com validacao em tempo real e automacao de formularios.

- Integracao **ViaCEP** para preenchimento automatico de endereco
- Sistema de **cupons de desconto**
- Validacao de campos em tempo real

**Stack:** HTML | CSS | JavaScript
